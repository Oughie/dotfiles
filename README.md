# Oughie's dotfiles

A catppuccin-themed setup for Arch Linux that includes:
bspwm, dunst, fastfetch, htop, kitty, lvim, neofetch (deprecated), picom, polybar, rofi, sxhkd, zsh, hyprland & waybar.
